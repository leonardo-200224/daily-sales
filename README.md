# Daily Sales Register

## Description

This project is a Python program that registers daily sales in a small store.

The system is designed to help store administrators organize their sales and calculate the total revenue of the day.

The program is simple and modular.

## How it works

1. The program asks the user to enter the product name, price, and quantity.
2. The system stores each sale in a dictionary inside a list.
3. The program repeats the process while the user wants to continue registering sales.
4. The system calculates the total revenue using the price and quantity of each product.
5. The program generates a final summary that shows the products sold and the total revenue.

Currently, the program is processing the sales data and generating the daily report.
